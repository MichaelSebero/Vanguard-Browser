## How to install 

### LINUX
 
. Overwrite your **[ .librewolf ]** folder which is found at **[ /home/$USER ]**

### WINDOWS

. Download the **[ .librewolf ]** file and extract **[ e2fxn5hu.default-release ]** files to -

**[ C:\Users\YOUR-USER\AppData\Roaming\Librewolf\Profiles\YOUR-PROFILE ]**
